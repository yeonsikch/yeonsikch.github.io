1. pip 다시 설치<br>
  - `curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py`<br>
  - `python3 get-pip.py --force-reinstall`<br>
<br>
2. pip 설치 확인<br>
  - `pip -V`<br>
  - `pip3 -V`<br>
<br>
3. pip 리스트 확인<br>
  - `pip list`<br>
  - `pip3 list`<br>
<br>
- pip는 python2.x 버전, pip3는 python3.x 버전.<br>
  - 단, conda와 같은 가상환경속에서는 pip 입력 시 conda 생성 시 지정해주었던 python 버전으로 매핑이 된다.
